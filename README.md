# PatternFly Finder Extension

A proof of concept for a macOS Finder-style column browser component, built as a [PatternFly Java](https://github.com/patternfly-java/patternfly-java/) extension.

## Demo

A live demo is available at https://hpehl.github.io/pf-finder-extension/
